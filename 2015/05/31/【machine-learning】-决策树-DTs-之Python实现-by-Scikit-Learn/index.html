安装 GraphViz

Graphviz (Graph Visualization Software) 是一个由AT&T实验室启动的开源工具包，用于绘制DOT语言脚本描述的图形。

  • GraphViz不能通过pip安装。 所以，下载对于Mac的正确版本(graphviz-2.36.0.pkg)之后，自行安装。
